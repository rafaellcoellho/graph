# Graph

A simple graph implementation in C using adjacency list for this [blog post](https://rafaellcoellho.netlify.com/2019/03/03/grafo-em-c/).

## Prerequisites

Just install [meson and ninja](https://mesonbuild.com/Getting-meson.html).

## Usage

Compile and run

```bash
$ meson build
$ cd build
$ ninja
$ ./src/main
```
